1) install django (pip install django)
2) create structure of project dirs and files (django-admin.py startproject mysite .)
3) change settings:
    set TZ can find in https://en.wikipedia.org/wiki/List_of_tz_database_time_zones
    settings.py TIME_ZONE = 'Europe/Moscow'
    set dir that content static resources STATIC_ROOT = os.path.join(BASE_DIR, 'static')
    ALLOWED_HOSTS = ['127.0.0.1', '<твоё_имя_пользователя>.pythonanywhere.com']
    set db settings
3) create db for project (python manage.py migrate)
4) start server (python manage.py runserver 127.0.0.1:8000)
5) check working
6) create new app in project (python manage.py startapp blog)
7) add application to settings (settings.py INSTALLED_APPS = [ ......, blog,]
8) add models
9) python manage.py makemigrations blog
10) python manage.py migrate blog
11) register models in admin.py
12) create a superuser (python manage.py createsuperuser)
13) can administrate now (http://127.0.0.1:8000/admin)
